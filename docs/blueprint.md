# **App Name**: Pravia Assistant

## Core Features:

- WhatsApp Integration: Directly open a WhatsApp chat with pre-filled message for new client inquiries.
- Contact Form Submission: Allow users to submit their information and query through a contact form.
- Automated Message Composition: Automatically composes a message that includes the client's details (name, phone number, query) to send to Pravia's WhatsApp.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) to evoke trust and professionalism. A direct reference to the color already used by Pravia in their branding.
- Background color: Dark slate gray (#0F172A), a desaturated indigo, provides contrast and a sophisticated feel. This reproduces an existing choice from the Pravia app, providing visual continuity.
- Accent color: Cyan (#22D3EE) for interactive elements and highlights, complementing the primary indigo.
- Body and headline font: 'Inter' sans-serif, chosen for its clean, modern appearance and excellent readability, as well as its neutral and objective tone.
- Use simple, outlined icons for navigation and key features.
- Maintain a clean, grid-based layout with clear sections for content and forms.
- Subtle transitions and hover effects for interactive elements to enhance user experience.
